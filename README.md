# Brew Haven Coffee Shop

**Author:** RealitiNova
**Project Type:** Static Website (HTML/CSS)  
**Repository:** Brew-Haven-Coffe-Shop

---

## Overview / Description
Brew Haven Coffee Shop is a simple, responsive, multi-page website for a coffee shop. The site provides a clean landing page experience, a menu page for browsing items, and a contact page where visitors can find contact details and reach out.

---

## How to Use / Functionality
- **Home (`index.html`)**: Landing page with key highlights and navigation.
- **Menu (`menu.html`)**: Displays coffee shop items and menu information.
- **Contact (`contact.html`)**: Provides contact details and a contact form layout for visitor messages.
- **Navigation**: Use the menu links in the header to move between pages.

To run locally:
1. Download or clone the repository.
2. Open `index.html` in your browser.

## User Stories

1. As a customer, I want to view the coffee menu so that I can decide what to order.

2. As a visitor, I want to see the contact page so that I can find the shop location and opening hours.

3. As a user, I want to navigate between pages easily so that I can browse the website smoothly.

4. As a customer, I want to see images of coffee so that I can choose what I like.
